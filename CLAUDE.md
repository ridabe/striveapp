# Instruções do Projeto — strivePersonalApp

## Graphify (obrigatório)

- **Antes** de alterar código, criar novas features ou fazer qualquer mudança relevante no projeto, consulte primeiro o Graphify (`/graphify`) para entender o contexto, arquitetura e relações existentes no código.
- **Depois** de qualquer alteração no código (edição, criação, refatoração, remoção de arquivos), rode `/graphify --update` automaticamente, sem precisar que o usuário peça. Isso vale para toda alteração feita nesta sessão ou em sessões futuras — é uma regra permanente do projeto.
